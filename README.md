# Odin Recipes

A basic recipe website built as part of The Odin Project Foundations course.

This project demonstrates fundamental HTML concepts, including:

- Creating and structuring web pages with HTML
- Using headings, paragraphs, lists, and images
- Creating links between multiple pages
- Organizing files and directories within a project

The website contains a collection of recipe pages, each with ingredients, instructions, and images. The goal of this project is to practice building a simple multi-page website using only HTML.

## Skills Demonstrated

- HTML boilerplate structure
- Relative and absolute links
- Working with images
- Lists (`ul`, `ol`)
- File organization
- Basic Git and GitHub workflow

## Technologies Used

- HTML5
