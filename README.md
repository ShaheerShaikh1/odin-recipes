# Odin Recipes 🍽️

This is a simple recipe website built as part of The Odin Project foundations curriculum.  
The project focuses on practicing basic HTML structure, links, and images.

## 📌 Features

- Home page with links to recipes
- Individual recipe pages
- Basic use of HTML elements like headings, lists, and images
- Internal linking between pages

## 🧱 Built With

- HTML5
